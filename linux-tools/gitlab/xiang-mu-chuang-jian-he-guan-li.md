# 项目创建和管理

* 每个用户都可以创建项目/组
* 普通用户创建的项目/组在其用户目录下
* 管理员创建项目/组在服务器根目录下

每个项目/组都具有三种级别的权限

| visibility | Level |
| :--- | :--- |
| Private | Private visibility has been restricted by the administrator. |
| Internal | The project can be accessed by any logged in user. |
| PublicThe | project can be accessed without any authentication. |
